Tasks Usability Score Improvements

Usability Study

This Google site was created by the Center for Civic Engagement at Davidson College. On the site, you will find resources that help professors better engage their curriculum and scholarship with the communities around them.

Task #1

Find the purpose of this website.

It took the user 15 seconds to find the text on the homepage that explains the website's purpose: "Curated by the Center for Civic Engagement, these internal and external resources aim to aid faculty in the knowledge-building, planning, designing, and implementing of community-based learning courses and programs, including strategies for reflection and assessment."

Task #2

Find the contact information for this website.

It took the user about two minutes to complete the task. The user tried to hit the 'i' icon, found a Google feedback form with no specific contacts listed, and then clicked directory. Eventually, he found the contacts in additional resources.

Task #3

Find example syllabi for an environmental science class.

The user was unable to locate resources for an environmental studies syllabus without using the search bar. First, the user clicked "Davidson-specific syllabus" under the fundamentals navigation tab, and then gave up and used the search bar.

Task #4

Find the list of community-engaged faculty fellows.

The user completed this task in around 5-10 seconds by scanning the menu options and finding it under the "More" tab.

Task #5

Find the list of readings about engaged scholarship.

The user was able to locate readings about engaged scholarship under the fundamentals navigation tab within five seconds.
